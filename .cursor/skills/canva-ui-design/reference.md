# Canva UI Design Reference

Extended color theory, font pairing examples, and grid specifications for the canva-ui-design skill.

## Color Theory (Extended)

### Color Temperature

- **Cool** (blue, green): Calming, professional, trust. Good for finance, healthcare, tech.
- **Warm** (red, orange, yellow): Energy, urgency, optimism. Good for food, entertainment, CTAs.
- **Neutral** (black, white, gray, beige): Versatile; pair with warm or cool for contrast.

### Value and Saturation

- **Value**: Lightness or darkness. High contrast (light vs dark) creates emphasis.
- **Saturation**: Vivid vs muted. Bright colors draw attention; desaturated colors recede.
- **Rule**: Use different values/saturations to establish hierarchy; similar values have similar visual weight.

### Gradients

- Gradients add depth; two or more tints/shades blend to suggest light/shadow.
- Use sparingly for backgrounds or accents; avoid clashing with text contrast needs.

## Font Pairing Examples

| Primary (headlines) | Secondary (body) | Use case |
|--------------------|------------------|----------|
| Serif (Playfair, Merriweather) | Sans-serif (Open Sans, Lato) | Editorial, formal |
| Sans-serif (Montserrat, Poppins) | Sans-serif (Source Sans, Inter) | Modern, clean |
| Display (Bebas, Oswald) | Sans-serif (Roboto) | Bold, marketing |
| Monospace (Fira Code, JetBrains Mono) | Sans-serif | Dev tools, technical |
| Rounded (Nunito, Quicksand) | Sans-serif | Playful, friendly |

**Rule**: Pair one distinctive font with one neutral, readable font.

## Grid Specifications

### Web (12-column)

- **Columns**: 12
- **Gutters**: 16–24px between columns
- **Margins**: 24–48px at viewport edges
- **Breakpoints** (typical): 640, 768, 1024, 1280, 1536px

### Mobile

- **Single column**: Full-width content with side padding (16–24px)
- **2-column**: Cards, lists; gutter 12–16px

### App Icons

- **iOS**: 1024×1024 (App Store), 180×180 (iPhone), 167×167 (iPad Pro)
- **Android**: 512×512 (Play Store), 192×192 (xxxhdpi)
- **Favicon**: 32×32 minimum; ico or PNG

## Symmetry and Asymmetry

- **Symmetrical**: Order, formality, balance. Mirrored layouts; good for landing hero sections.
- **Asymmetrical**: Dynamic, variety. Unbalanced layouts; good for emphasis and modern feel.
- **Tip**: Use symmetry for stability; add asymmetry to highlight a focal point.

## Transparency and Depth

- **Overlays**: Semi-transparent layers for frosted glass, modal backdrops.
- **Opacity**: Lower opacity for secondary elements; full opacity for primary.
- **Texture**: Subtle texture adds dimensionality; use with low opacity to avoid distraction.
